# 📋 작업 브리프

**원 명령:** 현재부터는 아이디어 확장이 아니라 실제 MVP 제작 단계로 전환한다.

Designer와 Writer는 협업하여
모바일 기준 실제 서비스 화면 흐름을 확정하라.

다음 화면들을 순서대로 설계하라:

1. 스플래시 화면
2. 문양 선택 화면
3. 질문 진행 화면
4. 감정 변화 기록 화면
5. 무료 결과 화면
6. 유료 전환 화면
7. 유료 심층 리포트 화면

각 화면마다 반드시 포함:

* 실제 UI 구성요소
* 버튼명
* 문구
* 사용자 행동
* 다음 화면 이동 조건
* 감정 유도 목적

추상적인 설명 금지.
실제 서비스 제작 가능한 수준으로 작성.

## 요약
아이디어 구체화 단계를 끝내고, 실제 MVP(최소 기능 제품) 제작 단계로 전환합니다. Designer와 Writer가 협업하여 사용자 경험 흐름에 맞춰 7단계의 모바일 서비스 화면별 상세 설계안을 작성해야 합니다.

## 분배
- **🎨 Designer**: 모바일 앱 기준으로, 스플래시부터 유료 심층 리포트까지 총 7개 화면에 대한 실제 UI/UX 디자인 브리프를 작성해 주세요. 각 화면별로 필요한 UI 요소(버튼, 이미지 배치 등), 와이어프레임 구조, 컬러 팔레트 적용 가이드라인을 포함해야 합니다.
- **✍️ Writer**: 디자이너가 제공한 7개 화면 흐름(스플래시 → 문양 선택 → 질문 진행 → 감정 변화 기록 → 무료 결과 → 유료 전환 → 심층 리포트)에 맞춰, 각 단계별 핵심 카피라이팅(문구), 버튼명, 사용자에게 전달할 메시지(CTA 포함)를 작성해 주세요. 특히 '감정 유도 목적'을 달성하기 위한 설득력 있는 문구를 강조해야 합니다.

## 📝 MOONCO 심층 테스트 MVP 카피라이팅 & 플로우 설계 (Writer Ver.)

**[작업 목표 재확인]**
디자이너가 정의한 7개 화면 흐름(스플래시 → 문양 선택 → 질문 진행 → 감정 변화 기록 → 무료 결과 → 유료 전환 → 심층 리포트)에 맞춰, **실제 MVP 제작 가능한 수준의 카피라이팅, 버튼명, 사용자 메시지**를 작성합니다. 핵심은 '감정 유도'와 '설득력 있는 CTA'입니다.

---

### 🖼️ Screen 1: 스플래시 화면 (Splash Screen)

*   **UI 구성요소:** MOONCO 로고 + 태그라인, 심플한 애니메이션 스피너
*   **핵심 카피/메시지:** "당신의 마음을 비추는 가장 깊은 질문." (태그라인)
*   **버튼명:** 없음 (자동 이동)
*   **사용자 행동:** 대기 (로딩 과정 인지)
*   **다음 화면 이동 조건:** 2~3초 경과 후 자동 전환.
*   **감정 유도 목적:** **호기심 자극.** '나의 무언가'를 발견할 것이라는 기대감을 심어주며, 서비스에 대한 첫인상을 신뢰 기반으로 구축.

### 🖼️ Screen 2: 문양 선택 화면 (Onboarding/Identity Selection)

*   **UI 구성요소:** 제목, 설명 텍스트, 3가지 옵션 버튼(문양 A, B, C), 뒤로가기 버튼
*   **제목 카피:** "어떤 질문에 가장 끌리시나요? 당신의 오늘 컨디션을 선택해주세요."
*   **설명 텍스트:** "당신이 지금 느끼는 감정 상태와 가장 유사한 '문양'을 골라주세요. 이 문양이 당신에게 던져질 첫 번째 심리 테스트의 테마가 됩니다."
*   **버튼 옵션 (예시):** [💡 막연한 불안함] / [✨ 에너지가 넘치는 설렘] / [☁️ 무기력하고 지친 느낌]
*   **사용자 행동:** 가장 현재 자신의 상태와 맞는 문양 1개 선택.
*   **다음 화면 이동 조건:** 옵션 버튼 중 하나를 탭할 때.
*   **감정 유도 목적:** **자기 객관화 및 몰입.** 사용자가 스스로의 감정을 정의하고 '선택'하는 과정을 통해 서비스에 대한 통제감과 참여도를 높임.

### 🖼️ Screen 3: 질문 진행 화면 (Question Flow)

*   **UI 구성요소:** 문제 제시 영역, 선택지 카드 (A/B/C), 진행률 표시줄 (Progress Bar)
*   **문제 카피 (예시):** "주말 아침에 눈을 떴을 때, 당신의 기분은 어떤가요?"
*   **선택지 문구:**
    *   A. 활기차게 하루를 시작하고 싶다. (긍정적 상태)
    *   B. 그저 흘러가도 괜찮다는 느낌이다. (중립/수용)
    *   C. 침대와 한 몸이 되어 시간이 멈췄으면 좋겠다. (부정적 감정)
*   **버튼명:** [선택하기] (또는 다음 문항으로 자동 이동 유도)
*   **사용자 행동:** 제시된 질문에 대해 가장 와닿는 선택지 하나를 고르고 '선택하기' 클릭.
*   **다음 화면 이동 조건:** 모든 질문 완료 시점까지 순차적 진행.
*   **감정 유도 목적:** **참여와 직관성.** 어렵거나 학술적인 느낌 대신, 일상 대화처럼 느껴지게 하여 심리 테스트에 대한 심리적 장벽을 낮춤.

### 🖼️ Screen 4: 감정 변화 기록 화면 (Emotion Logging)

*   **UI 구성요소:** '오늘의 나' 그래프/슬라이더, 키워드 태그 추가 공간, 저장 버튼
*   **제목 카피:** "테스트를 마치고, 오늘 당신의 마음을 한 번 더 그려볼까요?"
*   **메시지 (설득 강화):** "심리 테스트는 시작일 뿐입니다. 가장 중요한 건 '나' 자신과의 대화죠. 지금 이 순간 느낀 미묘한 감정까지 기록하면 분석 정확도가 높아져요."
*   **슬라이더/입력:** 1~5점의 감정 지수 (예: 안정감, 활력도)와 사용자가 직접 키워드 입력 (예: '몽롱함', '미지근함').
*   **버튼명:** [나만의 기록 남기기]
*   **사용자 행동:** 테스트 결과가 끝난 후, 자발적인 회고를 통해 데이터를 추가 입력.
*   **다음 화면 이동 조건:** 데이터 저장 및 다음 단계(결과 확인)로 이동 준비 완료 시.
*   **감정 유도 목적:** **소속감/지속성 부여.** 단순한 '퀴즈'가 아닌, '나의 일상 기록 관리 도구'라는 인식을 심어주어 서비스 이탈을 방지하고 다음 단계(결과 확인)에 대한 기대치를 높임.

### 🖼️ Screen 5: 무료 결과 화면 (Free Result Gate)

*   **UI 구성요소:** 테스트 요약 점수/키워드, '간단 분석' 섹션, CTA 버튼
*   **제목 카피:** "당신의 현재 상태는 [OOO]에 가깝습니다." (결과 제시)
*   **핵심 메시지 (가치 전달):** "표면적인 이해만으로도 큰 위로가 되셨을 거예요. 하지만 진짜 근본 원인까지 알기 위해서는 조금 더 깊은 탐구가 필요해요."
*   **무료 제공 내용:** '현재 상태에 대한 일반론적 해석' (예: 피로 누적으로 인한 일시적 현상)
*   **CTA 버튼:** **[✨ 10분 만에 근본 원인 찾기 (유료 전환)]** (가장 눈에 띄는 색상 사용)
*   **대체 CTA:** [다음에 다시 테스트해보기] (이탈 방지용, 중요도 낮음)
*   **사용자 행동:** 무료 결과 확인 후, 더 깊은 분석을 위해 유료 버튼 클릭.
*   **감정 유도 목적:** **불완전함(Curiosity Gap) 조성.** '일부만 알려주고 핵심적인 답은 돈으로 결제하게 만듦'으로써, 사용자가 스스로 지불 가치를 느끼게 함.

### 🖼️ Screen 6: 유료 전환 화면 (Payment Gate / Upsell)

*   **UI 구성요소:** 문제 제기(Pain Point), 해결책 제시(Solution), 가격 정보, 결제 버튼
*   **제목 카피 (공감대 형성):** "혹시 '이건 왜 이럴까?'라는 막연한 질문에 답답하신가요?"
*   **문제점 강조:** "단순 테스트 결과는 현상만 보여줍니다. **원인과 해결책까지 알고 싶다면, 전문적인 관점이 필요합니다.**"
*   **솔루션 제시 (가치 극대화):** "MOONCO 심층 리포트는 당신의 패턴을 분석하여, 구체적인 '행동 변화 가이드'와 맞춤형 시각 자료를 제공합니다."
*   **가격/혜택:** [심층 리포트] ₩15,000 (오늘만 특별 할인가!)
*   **버튼명:** **[🚀 심층 분석 받기 및 결제하기]**
*   **사용자 행동:** 문제 해결에 대한 니즈가 강해져 '구매'를 결정하고 결제 진행.
*   **감정 유도 목적:** **불안 해소 약속.** 사용자가 느끼는 모호한 불안(Pain)을 명확한 지식/행동 가이드(Gain)로 치환하여, 비용 지출을 '필수적인 투자'로 인식하게 만듦.

### 🖼️ Screen 7: 유료 심층 리포트 화면 (Final Deliverable)

*   **UI 구성요소:** 최종 보고서 요약 페이지 (핵심 인사이트 3가지), 행동 플랜 제시, 다운로드/공유 버튼
*   **제목 카피:** "✨ [OOO] 패턴 분석 리포트: 당신의 마음 설계도"
*   **핵심 메시지 (권위 부여):** "이 보고서는 단순한 결과지가 아닙니다. 다음 단계로 나아가기 위한 **당신만을 위한 맞춤형 로드맵**입니다."
*   **인사이트 구조:**
    1.  **진단:** 당신의 현재 상태와 핵심 결핍 영역. (전문 용어 사용으로 권위 부여)
    2.  **원인 분석:** 이 패턴이 생겨난 근본적인 심리적 배경.
    3.  **액션 플랜:** 7일 동안 실천 가능한 구체적인 행동 가이드 3가지. (가장 실용적이어야 함)
*   **버튼명:** [보고서 다운로드 및 저장] / [친구에게 공유하고 함께 성장하기]
*   **사용자 행동:** 리포트 내용을 검토하고, 제공된 로드맵을 따라 생활 변화를 시도함.
*   **감정 유도 목적:** **성취감과 지속적 관계 유지.** 결제에 대한 만족감을 '실질적인 삶의 개선'으로 느끼게 하고, 추후 재방문 및 다른 서비스 이용에 대한 기대치를 최고치로 설정.

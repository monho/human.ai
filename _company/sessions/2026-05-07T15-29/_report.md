### 🥇 TVST (Top Value Single Task): 심리 테스트 MVP '결과 해석 시퀀스' 완성 및 검증
*   **목표:** 사용자가 가장 감정적으로 몰입하는 지점인 '최종 결과 리포트/해석'의 스토리텔링 흐름을 최종 확정하고, 이 흐름에 필요한 데이터 포맷까지 명확히 합니다.
*   **필요 에이전트 조합:** Writer ✍️ + Designer 🎨 + Developer 💻

#### ✅ 구체적인 다음 액션 아이템 (Next Action Item) 지시 사항:

1.  **[Writer] 최종 코칭 메시지 및 스토리라인 확정:**
    *   Designer가 제공한 3종 Mockup을 바탕으로, '결과 해석' 섹션에 들어갈 **톤앤매너가 일관된 핵심 문구(Headline/Body Copy)** 초안을 5가지 버전으로 작성해 주세요. (진단 금지, 성장 안내서 느낌 유지)
    *   **필수:** 이 문구들이 어떤 사용자 경험 시퀀스(예: 테스트 시작 → 문제 인지 → 전환점 도달 → 최종 결론)에서 나오는지 스토리보드와 함께 매핑하세요.

2.  **[Designer] 와이어프레임 기반의 정보 구조 확정:**
    *   Writer가 제공할 5가지 버전의 메시지를 가장 효과적으로 담을 수 있는 **'정보 계층 구조(Information Hierarchy)'** 를 최종 결정해 주세요.
    *   어떤 문구가 어떤 시각적 요소(카드, 그래프, 섹션 제목 등)로 표현될지 명확한 가이드라인을 재차 제시해 주시고, 특히 '성장 과정 기록' 톤에 맞는 레이아웃 예시를 제공해주세요.

3.  **[Developer] 데이터 매핑 및 API Mockup 최종화:**
    *   Writer가 작성할 문구와 Designer가 설계할 구조에 맞춰, **실제 사용자 입력값(Input Data)이 이 콘텐츠 요소들에 어떻게 반영되어야 하는지**를 정의해 주세요.
    *   '심층 패턴 분석 리포트'의 핵심 데이터 포인트 3가지와 그 데이터를 표현하기 위한 **최소한의 API 응답 구조(Mockup)** 를 최종 확정해야 합니다.
